# Offert Apps Projekt
Under detta projekt används AI, (ChatGPT), för kodnings stöd/hjälp.
## Projekt Idé
Tanken med detta projekt är att användaren ska snabbt och enkelt kunna göra en offert för att sedan kunna skriva ut, mejla eller skicka den via appen.
# Projekt Faser
Nedan är en liten roadmap för att lättare organisera projektet och hålla reda på vart projektet är just nu.
